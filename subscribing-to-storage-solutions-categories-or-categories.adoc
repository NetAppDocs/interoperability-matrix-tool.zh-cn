= 订阅存储解决方案、类别或组件
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
您可以订阅您选择的单个或多个存储解决方案、类别或组件。

*订阅*选项卡允许您订阅以下订阅：

* 订阅存储解决方案
* 正在订阅类别
* 订阅组件
